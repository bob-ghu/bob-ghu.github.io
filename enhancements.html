Enhancements

Color-Changing Text

- When the cursor hovers over the company name (HYPERTECH) on the Navigation Menu, it constantly changes color.

- It provides a visual indicator that the hovered text can be clicked.

- This is achieved by using [@keyframes] and the property [animation].

- @keyframes allows you to define and control animations by specifying keyframes at various points in an animation's timeline.

- By specifying a color at each point (10%, 20%, etc), the animation will transition through the colours smoothly.

- The property [animation] helps control the speed and duration of the animation

- Since the Navigation Menu is fixed , there is no hyperlink provided as the company name (HYPERTECH) is always on the top left.

- In the CSS files, the CSS rules can be found at line 49-79.

Image/Text Fade In Animation

- Whenever a page is loaded, an animation starts whereas specific text/images fade into view.

- This gives more focus onto those text/images which have important information.

- This is achieved by using [@keyframes] and the properties [animation, transform].

- @keyframes allows you to define and control animations by specifying keyframes at various points in an animation's timeline.

- By specifying the position and opacity at each point (10%, 20%, etc), the animation will transition through them smoothly.

- The property [animation] helps control the speed and duration of the animation

- The property [transform] sets the starting and ending positions of the keyframes

- Every page including this one has the fade in animation. Links to the others [I,II,III,IV]

- In the CSS files, the CSS rules can be found at line 116-132.
