JavaScript Enhancements

Navigation Underline Display

- When the user navigates through the website using the Navigation Bar, the navigation text of the current page will be underlined.

- This is achieved by using [document.title] to check the user's currently viewed page and using [.style.textDecoration] to change the styling of the navigation text.

- Every page including this one has the dynamic display. Links to the others [I,II,III,IV,V]

Form Timer

- When the user navigates to the Application page to apply for a job, a timer will be started for that session.

- If the form is not completed and submitted before the timer ends, the user will be redirected to the Home page.

- The user will have to re-fill the form again if the aforementioned case occurs.

- This is achieved by using [setInterval] which is set to perform a function at every 1000ms (= 1 second).

- Every interval the time display will update using [.textContent] and the format is created using [string.padStart] (05:00-0:00)

- The timer is displayed in the Application page.

Job Details Changing Opacity

- In the Jobs page, when the user clicks on either the Usability Specialist or Web Developer picture, the non-relevant job details will change opacity.

- For example, if the Usability Specialist picture is clicked, the Web Developer job details will change opacity and become faded.

- This allows the user to focus on only the relevant information.

- This is achieved by using [.addEventListener] to register a click on the image, [.querySelectorAll] to select all child elements of the job details section and [.style.opacity] to change the selected elements' opacity.

- This is implemented in the Jobs page.
