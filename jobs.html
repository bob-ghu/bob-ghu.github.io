USABILITY SPECIALIST

Job Title:

Usability Specialist

Salary Range:

RM7,250 - RM12,000

Reports to:

Director of User Experience (UX)

Brief Description of the Position:

The Usability Specialist plays a crucial role in ensuring that our digital products provide exceptional user experiences. This position involves evaluating and optimizing the usability, accessibility, and overall user satisfaction of our software applications and websites.

Key Responsibilities:

  1. Conduct usability studies, user testing, and heuristic evaluations to identify usability issues and gather insights.
  2. Collaborate with cross-functional teams to design, implement, and improve user interfaces that align with user needs and industry best practices.
  3. Analyze user feedback and data to make informed recommendations for UX improvements.
  4. Create and maintain usability reports, user personas, and user journey maps.
  5. Work closely with designers and developers to ensure usability considerations are integrated into the development process.
  6. Conduct accessibility assessments to ensure compliance with WCAG guidelines.
  7. Stay up-to-date with industry trends and emerging best practices in usability and UX design.

Required Qualifications, Skills, Knowledge, and Attributes:

Essential:

  • Bachelor's degree in Human-Computer Interaction, Psychology, or a related field.
  • 3+ years of experience in usability testing, user research, or a related role.
  • Proficiency in usability evaluation methods and tools.
  • Strong analytical and problem-solving skills.
  • Excellent communication and presentation skills.
  • Familiarity with UX/UI design principles.
  • Ability to work independently and as part of a team.

Preferable:

  • Master's degree in Human-Computer Interaction or a related field.
  • Experience with usability testing software (e.g., UsabilityHub, UserTesting).
  • Certification in usability or UX design.
  • Knowledge of web accessibility standards (WCAG).
  • Familiarity with prototyping tools (e.g., Sketch, Figma).
  • Experience in conducting remote usability studies.

WEB DEVELOPER

Job Title:

Web Developer

Salary Range:

RM9,200 - RM12,000

Reports to:

Lead Web Developer

Brief Description of the Position:

The Web Developer is responsible for designing, developing, and maintaining web applications and websites. This role involves creating responsive and visually appealing user interfaces, writing clean and efficient code, and collaborating with cross-functional teams to deliver high-quality web-based solutions.

Key Responsibilities:

  1. Design and develop user-friendly and visually engaging web interfaces.
  2. Write clean, efficient, and maintainable code for web applications and websites.
  3. Collaborate with designers to implement responsive designs that work seamlessly across devices.
  4. Integrate data from various back-end services and databases.
  5. Optimize web applications for speed, performance, and scalability.
  6. Perform code reviews and provide constructive feedback to team members.
  7. Troubleshoot and resolve issues reported by users or QA testers.
  8. Stay up-to-date with the latest web development trends and best practices.
  9. Participate in project planning and execution.

Required Qualifications, Skills, Knowledge, and Attributes:

Essential:

  • Bachelor's degree in Computer Science, Web Development, or a related field.
  • 2+ years of experience in web development using HTML, CSS, and JavaScript.
  • Proficiency in front-end frameworks such as React, Vue.js, or Angular.
  • Strong problem-solving and debugging skills.
  • Familiarity with version control systems (e.g., Git).
  • Ability to work both independently and collaboratively in a team.
  • Effective communication skills.

Preferable:

  • Experience with back-end programming languages (e.g., Node.js, Python, Ruby, PHP).
  • Knowledge of RESTful API design and integration.
  • Familiarity with responsive and mobile-first web design principles.
  • Experience with web development frameworks (e.g., Express, Ruby on Rails).
  • Understanding of web security best practices.
  • Proficiency in build tools and package managers (e.g., Webpack, npm).
